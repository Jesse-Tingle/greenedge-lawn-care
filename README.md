# GreenEdge Lawn Care

GreenEdge Lawn Care is a demo website for a fictional lawn care business in North Central Indiana.

This project was created as a sample freelance web development project for small local service businesses that need a clean, professional online presence.

## Live Site

https://jesse-tingle.github.io/greenedge-lawn-care/

## Project Purpose

The goal of this project is to demonstrate a simple one-page business website that helps a lawn care company:

- Look more professional online
- Clearly show services
- Display the areas they serve
- Build trust with reviews
- Make it easy for customers to call or request a quote

## Features

- Mobile-friendly layout
- Hero section with call-to-action buttons
- Services section
- Service area section
- Recent work section
- Customer reviews section
- Quote request form with basic JavaScript success message
- Click-to-call phone buttons
- Basic SEO title and meta description

## Built With

- HTML
- CSS
- JavaScript
